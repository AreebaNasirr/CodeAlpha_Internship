# CodeAlpha C++ Internship Tasks

This repository contains my solutions for the **CodeAlpha C++ Programming Internship**. All tasks are implemented using **Object-Oriented Programming (OOP)** concepts in C++ and demonstrate problem solving, file handling, and class-based design.

## Projects Included

## 1. CGPA Calculator
A C++ application that calculates a student's CGPA based on course grades and credit hours.
**Features:**
- Input multiple courses
- Grade point calculation
- Credit hour management
- CGPA calculation
- OOP-based implementation

## 2. Login & Registration System
A simple user authentication system that allows users to register and log in using file handling.

**Features:**
- User registration
- Login authentication
- Duplicate username checking
- Input validation
- Credentials stored in a text file

## 3. Banking System
A menu-driven banking application developed using Object-Oriented Programming.

**Features:**
- Customer and account management
- Deposit funds
- Withdraw funds
- Transfer money between accounts
- Balance inquiry
- Transaction history
- File handling for data storage

## Technologies Used

- C++
- Object-Oriented Programming (OOP)
- File Handling
- Standard Template Library (STL)
